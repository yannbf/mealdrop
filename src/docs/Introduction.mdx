# Welcome to Chromatic Demo

This Storybook is based on the [Mealdrop app](https://github.com/yannbf/mealdrop), a project used in the [Storybook for React apps](https://www.newline.co/courses/storybook-for-react-apps) course by [Yann Braga](https://twitter.com/yannbf).

### Examples

- [Header](?path=/story/components-header--default) is a good example of using viewports to test responsiveness
- [RestaurantCard](?path=/story/components-restaurantcard--default) demonstrates testing light & dark theme variants using [modes](https://www.chromatic.com/docs/modes/)
- [Home](?path=/story/userflows-app--home) is a basic example of a story that mocks API requests using [MSW](https://storybook.js.org/docs/writing-stories/mocking-network-requests)
- [RestaurantDetailPage](?path=/story/pages-restaurantdetailpage--with-modal-open) is an example of a full page story with [interaction tests](https://www.chromatic.com/docs/interactions/) and mocked API requests
- [App Â» ToCheckoutPage](?path=/story/userflows-app--to-checkout-page) is another full page example with complex (multi-page) interactions and mocked API requests
